# 🐍 Belajar Python Seru

Kumpulan studi kasus Python interaktif untuk anak SD-SMP, dikemas dalam bentuk
notebook (`.ipynb`) yang bisa langsung dijalankan sambil belajar. Setiap project
punya penjelasan bertahap, contoh kode, mini game, dan tantangan di akhir.

## 🎯 Tujuan

Membuat belajar Python jadi lebih menarik lewat permainan dan simulasi sederhana
yang dekat dengan keseharian anak-anak, sambil tetap mengajarkan konsep dasar
pemrograman secara bertahap.

## 📚 Daftar Project

| No | Project | Konsep yang Dipelajari |
|----|---------|--------------------------|
| 1 | [Tebak Angka](01_tebak_angka.ipynb) | variabel, `random`, `input`, `if-else`, `while` |
| 2 | [Kalkulator Cerita Matematika](02_kalkulator_cerita.ipynb) | f-string, list & dictionary, fungsi |
| 3 | [Kuis Interaktif](03_kuis_interaktif.ipynb) | list of dictionary, `enumerate`, kondisi bertingkat |
| 4 | [Hangman - Tebak Kata](04_hangman.ipynb) | string manipulation, `set`, list comprehension |
| 5 | [Simulasi Toko Kelontong](05_toko_kelontong.ipynb) | dictionary, format angka, validasi input |
| 6 | [Turtle Graphics](06_turtle_graphics.ipynb) | modul `turtle`, perulangan, fungsi gambar |
| 7 | [Petualangan Teks](07_petualangan_teks.ipynb) | dictionary bersarang, state machine sederhana |
| 8 | [Pembuat & Pengecek Kode Rahasia](08_password_generator.ipynb) | modul `string`, `random.choice`, validasi |
| 9 | [Kalender Ulang Tahun](09_kalender_ultah.ipynb) | modul `datetime`, perhitungan tanggal |

Setiap project bisa dikerjakan berurutan dari yang paling mudah (1) ke yang lebih
menantang (9), tapi juga bisa dikerjakan bebas sesuai minat.

## 🚀 Cara Menjalankan

1. Clone atau download repo ini.
2. Buka salah satu file `.ipynb` menggunakan:
   - [Jupyter Notebook/Lab](https://jupyter.org/install) di komputer, atau
   - [Google Colab](https://colab.research.google.com/) (tinggal upload filenya)
3. Jalankan setiap sel kode secara berurutan dari atas ke bawah (`Shift + Enter`).

> ⚠️ Khusus project **Turtle Graphics**, jalankan di Jupyter yang terinstall di
> komputer (bukan Google Colab), karena modul `turtle` membuka jendela gambar
> terpisah yang tidak didukung Colab secara langsung.
